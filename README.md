# React Chair Store

A modern and responsive furniture store built with React and Vite.

## Features

- Responsive Design
- React Router
- Product Categories
- Modern UI
- Reusable Components

## Technologies

- React
- Vite
- JavaScript
- CSS

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
